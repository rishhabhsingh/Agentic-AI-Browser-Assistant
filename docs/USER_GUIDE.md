## üóÇÔ∏è Smart Tab Manager

### What It Does
Helps you manage browser tabs using AI to detect duplicates and suggest cleanup.

### How to Use

#### Method 1: AI Analysis (Recommended)
1. Open multiple tabs (the more, the better for testing)
2. Click BrowserBuddy ‚Üí **Manage Tabs**
3. Click **ü§ñ AI Analysis**
4. Wait 2-5 seconds
5. Review AI suggestions (highlighted in yellow)
6. Select tabs to close (or use auto-selected)
7. Click **üóëÔ∏è Close X Tab(s)**
8. Confirm and done!

#### Method 2: Quick Scan (Faster)
1. Click BrowserBuddy ‚Üí **Manage Tabs**
2. Click **üîç Quick Scan**
3. Instant results (rule-based, no AI)
4. Close tabs as needed

#### View Statistics
1. Click **üìä Stats**
2. See:
   - Total tabs open
   - HTTPS vs HTTP breakdown
   - Top 5 domains
   - Tab distribution

### Features Explained

**AI Analysis:**
- Uses Groq AI (Llama 3.3)
- Understands context and content
- Finds similar (not just exact) duplicates
- Provides confidence scores
- Smarter but slower (2-5 seconds)

**Quick Scan:**
- Rule-based detection
- Finds exact duplicates
- Detects blank/error pages
- Instant results
- Perfect for quick cleanup

**Tab List:**
- ‚úÖ Green highlight = Suggested to close
- üìå "Active" badge = Current tab
- Click checkbox to select/deselect
- Click tab to toggle selection

### Tips
- ‚úÖ Run AI Analysis weekly for deep cleanup
- ‚úÖ Use Quick Scan daily for maintenance
- ‚úÖ Don't close tabs with "Active" badge
- ‚úÖ Review suggestions before closing

## üîß Troubleshooting

### Common Issues

#### Extension Not Working

**Symptoms:** Icon grayed out, features don't load

**Solutions:**
1. Refresh the webpage (F5)
2. Reload extension:
   - Go to `chrome://extensions/`
   - Click refresh icon on BrowserBuddy
3. Restart Chrome
4. Reinstall extension if persist

#### Backend Offline

**Symptoms:** "‚ùå Backend Offline" message

**Solutions:**
1. Check if backend is running locally
2. For deployed version, check server status
3. Features with fallbacks still work
4. Contact support if persists

#### Form Filler Not Detecting Fields

**Symptoms:** "No form fields detected"

**Solutions:**
1. **Refresh the page first** (most common fix)
2. Make sure form is visible on screen
3. Try clicking inside a form field first
4. Some complex forms may not work
5. Check browser console for errors

#### AI Features Slow

**Symptoms:** Takes >10 seconds to respond

**Solutions:**
1. Check internet connection
2. Groq API might be busy (try again)
3. Use fallback features (Quick Scan)
4. Reduce content length
5. Try during off-peak hours

#### Resume Download Not Working

**Symptoms:** Can't download generated resume

**Solutions:**
1. Check if popup blocker is on
2. Try right-click ‚Üí "Save link as"
3. Check Downloads folder
4. Ensure backend is running
5. Try generating again

#### Bookmarks Not Saving

**Symptoms:** Bookmarks disappear after refresh

**Solutions:**
1. Check MongoDB connection
2. Check browser console for errors
3. Ensure you clicked "Save"
4. Check storage permissions
5. May need to enable MongoDB

## ‚ùì FAQ

### General

**Q: Is BrowserBuddy AI free?**
A: Yes! 100% free, forever. No hidden costs, no subscriptions.

**Q: Does it work offline?**
A: Partially. AI features need internet. Reading Mode, Form Filler (with saved profile) work offline.

**Q: Is my data safe?**
A: Yes. Data stays local or in your private MongoDB. We don't collect or sell data.

**Q: How does the AI work?**
A: We use Groq's Llama 3.3 model (free tier). Very fast and accurate.

**Q: Can I use it on mobile?**
A: Not yet. Chrome extensions only work on desktop browsers.

### Features

**Q: Why can't I get YouTube transcripts?**
A: YouTube API packages have issues. Feature is marked BETA/experimental for now.

**Q: Can Form Filler work on all forms?**
A: 85-90% of standard forms. Very complex or custom forms may not work.

**Q: Does Resume Optimizer actually work with ATS?**
A: Yes! We generate .docx files with ATS-friendly formatting and keywords.

**Q: How many bookmarks can I save?**
A: No limit! Depends on your MongoDB storage.

**Q: Can I export my data?**
A: Currently no export feature. Coming soon!

### Technical

**Q: What's the Chrome version requirement?**
A: Chrome 120+ (released late 2023)

**Q: Does it work on all websites?**
A: Most websites. Some (like chrome:// pages) are restricted by Chrome.

**Q: How much data does it use?**
A: Minimal. AI requests are text-only, very small.

**Q: Can I use my own AI API key?**
A: Yes! In settings, you can use OpenAI key instead of Groq.

**Q: Is the code open source?**
A: Yes! Check GitHub repository.

### Privacy & Security

**Q: Do you track my browsing?**
A: No. Analytics are OFF by default and are anonymous if enabled.

**Q: Where is my data stored?**
A: Locally in Chrome or in your private MongoDB instance.

**Q: Can you see my Form Filler data?**
A: No. It's stored locally or in your database. We have no access.

**Q: Do you sell my data?**
A: Never. We don't even collect it.

**Q: Is it safe to use on banking sites?**
A: Yes, but we don't recommend using Form Filler on banking sites for security.
