# BrowserBuddy AI  
## Professional Installation Guide (Google Chrome)

Thank you for installing **BrowserBuddy AI**. Please follow the instructions below to complete the installation successfully.

---

## System Requirements

- Latest version of Google Chrome  
- Desktop operating system:
  - Windows 10 or later  
  - macOS 11 or later  
  - Linux (latest stable distribution)  
- Active internet connection  

---

## Installation Steps

### Step 1: Download the Extension Package

1. Click the download link provided.
2. Save the `.zip` file to your **Downloads** folder (or a known location on your computer).

---

### Step 2: Extract the ZIP File

You must extract the contents before installing.

#### Windows
1. Right-click the ZIP file  
2. Select **Extract All**  
3. Choose a destination folder  
4. Click **Extract**

#### macOS
- Double-click the ZIP file to automatically extract it.

#### Linux
- Right-click the ZIP file  
- Select **Extract Here** or **Extract to‚Ä¶**

---

### Step 3: Open Chrome Extensions Page

1. Open **Google Chrome**
2. In the address bar, type:
    chrome://extensions/

3. Press **Enter**

---

### Step 4: Enable Developer Mode

1. Locate the **Developer mode** toggle in the top-right corner.
2. Turn it **ON**.

Additional options will appear once enabled.

---

### Step 5: Load the Extension

1. Click the **Load unpacked** button.
2. Navigate to the extracted **BrowserBuddy AI** folder.
3. Select the folder named `dist`.
4. Click **Select Folder** (Windows) or **Open** (macOS).

The extension should now appear in your Extensions list.

---
